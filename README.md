## Simple CLI (command line interface) to play tic-tac-toe

Clone the repo, cd into it, then:

```
$ npm install
...
$ node_modules/.bin/gulp
...
$ node lib/play.js
```

You'll need [node.js](https://nodejs.org/en/).
